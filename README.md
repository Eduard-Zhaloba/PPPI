# PPPI Calculator

Це командний проєкт, присвячений створенню тригонометричного калькулятора на C++.

## Суть проєкту
Мета проєкту — реалізувати калькулятор, здатний обчислювати основні тригонометричні функції:
- Синус (sin)
- Косинус (cos)
- Тангенс (tan)
- Котангенс (ctg)

Користувач може виконувати різні математичні дії над тригонометричними функціями
